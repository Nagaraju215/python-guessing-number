# 🎯 Number Guessing Game (Python)

A simple Python project where the computer selects a secret number and the player has to guess it within a limited number of attempts.

---

## 📌 Features

* Random number generation
* Limited number of attempts
* User input handling
* Instant feedback (Correct / Wrong)
* Beginner-friendly logic using loops and conditions

---

## 🧠 How the Game Works

1. The program generates a secret number (e.g., between 1 and 10).
2. The player gets a fixed number of chances (e.g., 3 attempts).
3. After each guess:

   * If correct → You win 🎉
   * If wrong → Try again
4. If attempts are over → Game ends          print("You guessed the wrong number!")
